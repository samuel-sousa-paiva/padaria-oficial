# Padaria Pão Quente — Starter (Módulos 11-13)

Projeto inicial para o trabalho em grupo do curso Web Design Híbrido.

## O que já está pronto
- HTML semântico (header, nav, main, section, article, footer)
- Acessibilidade (skip-link, aria-labels, foco visível, aria-live, prefers-reduced-motion)
- CSS responsivo com tokens (cores, raio, sombras)
- JS com contador de pedido e validação de formulário acessível

## O que vocês precisam fazer
1. Personalizar textos, imagens e cardápio reais do nicho
2. Adicionar pelo menos 1 imagem por card (com `alt` descritivo)
3. Aplicar Scrum/Kanban — usar a planilha disponibilizada pelo professor
4. Fazer commits semânticos no GitHub (feat, fix, docs, style)
5. Publicar via GitHub Pages

## Como abrir
Basta abrir `index.html` no navegador, ou usar Live Server no VS Code.

Bom projeto! 💻
